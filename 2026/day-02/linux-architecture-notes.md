###Linux Architecture, Processes, and systemd###

**Kernel** - Its core component of linux, Managing memory and other hardware componenent ensures everything runs smoothly, 
it handles task running programs & schedule system processes.
**UserSpace** - Where user give a command to computer to run the application or program.
**SystemD** - System is background service runs on linux, responsible for managing system resources, runs process id with PID1
**Process** - Process is simply command running on Linux OS whenever you enter the command operating system creates the process id 
to run that command.

#################Below are the commands to manage processes#######################
$ ps - to check the process 
$ kill <PID> - kill the process using PID
$ df -h - check disk space
$ free - check memory
$ top - check all the inteactive processes with hardware usage
$ htop - check all the inteactive processes with hardware usage
