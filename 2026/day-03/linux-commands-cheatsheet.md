------------------------Linux Cheat sheet-----------------------------------------
This cheat sheet covers Process management, file system & networking troubleshooting 
**Process Management-** 
__________________________________________________________________________________________________________________
_______Command________|_________________________Description_______________________________________________________
        ps            |                        Display information about current active process
______________________|___________________________________________________________________________________________        
        top           |                  Display information about real time view of running system
                      |                  it display system summary information as well as list of processes
______________________|___________________________________________________________________________________________
        htop          |     it is similar to top command but allows you to scroll vertically and horizontally,
                      |     you can observe all processes running on system
