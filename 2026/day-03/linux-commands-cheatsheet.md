<h1>Linux command cheatsheet</h1>

<h3>Process management </h3>
ps - check the system process
free -m - used to check the available memory
df -h - used to check disk space
top - view hardware usage & system processes in interactive mode
htop - view modern way hardware usage & system processes in interactive mode

<h3>File system</h3>
cat - view the file content
touch - create the empty file
ls - list file or directory
mkdir - create the new directory
rm - remove the file
cp - copy file or directory
pwd - present working directory
mv - move/rename the file/directory

<h3>Networking troubleshooting </h3>
ping - check the connectivity
ip addr - view the machine ip address
traceroute -view the route packets and also trace to network host
curl - fetches the webpage content
wget - download webpage the content to local
